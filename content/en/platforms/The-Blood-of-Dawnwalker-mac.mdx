## Mac Setup Checklist for Smooth Play (Cloud or Remote)

For **The Blood of Dawnwalker mac** planning, your network quality matters almost as much as GPU power when using cloud/remote methods.

### Network targets you should aim for

| Metric | Minimum | Recommended | Why It Matters |
|---|---|---|---|
| **Download speed** | **25 Mbps** | **50+ Mbps** | Supports stable high-res stream |
| **Upload speed** | **10 Mbps** | **20+ Mbps** | Important for remote control response |
| **Latency (ping)** | **&lt;40 ms** | **&lt;20 ms** | Lower input delay in combat |
| **Connection type** | Wi-Fi 5 | **Ethernet or Wi-Fi 6/6E** | Reduces jitter/frame drops |

### Step-by-step: cloud route

1. Confirm the game is available on your preferred cloud platform.
2. Use a wired Ethernet connection where possible.
3. Connect a controller directly to your Mac (USB or low-latency Bluetooth).
4. Set stream quality to “balanced” first; raise quality after testing combat.
5. Disable heavy background downloads and sync apps.
6. Test a 15-minute combat-heavy session before committing to long play.

### Step-by-step: remote PC route

1. Install remote streaming software on the host Windows PC.
2. Update GPU drivers and set the game profile to stable settings.
3. Connect host PC via Ethernet.
4. Keep Mac client on 5 GHz/6 GHz Wi-Fi or Ethernet adapter.
5. Limit host render resolution if stutter appears.
6. Tune bitrate and frame rate until controls feel immediate.
